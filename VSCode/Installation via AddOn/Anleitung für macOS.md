
# Installationsanleitung für macOS

## Schritt 1:

Öffnen Sie in Ihrem Webbrowser folgenden Link: https://code.visualstudio.com/

Sie sollten nun folgende Ansicht haben:

<img width="1115" alt="Bild1" src="https://user-images.githubusercontent.com/114981111/222764098-b3cc7c5d-c95b-46bb-a441-bac1059ead11.PNG">

## Schritt 2:

Klicken Sie auf die Schaltfäche, auf die der rote Pfeil zeigt.

<img width="1115" alt="Bild1_1" src="https://user-images.githubusercontent.com/114981111/222764490-47ed46f3-70ac-4388-a9cc-00faea2b0810.png">

Es öffnet sich ein Dropdown-Menü. Laden Sie den Stable Universal für macOS herunter.

<img width="1115" alt="Bild2" src="https://user-images.githubusercontent.com/114981111/222764992-d1579007-1074-4992-8384-521ece274285.PNG">

## Schritt 3

In Ihrem Download-Verzeichnis sollte sich nun folgende ZIP-Datei befinden:

<img width="136" alt="Bildschirm­foto 2023-03-04 um 12 22 11" src="https://user-images.githubusercontent.com/114981111/222898600-42fa2ba6-9e06-4b33-88a8-95138f067d00.png">

Klicken Sie auf Diesen und auf Ihrem Desktop sollte nun folgendes Icon erscheinen:

<img width="138" alt="Bildschirm­foto 2023-03-04 um 12 23 15" src="https://user-images.githubusercontent.com/114981111/222898636-c8bbe597-7c8e-48b8-b23f-bf3f1da64d7f.png">

Starten Sie Visual Studio Code in dem Sie auf das Icon klicken.

## Schritt 4

Sie sollten nun folgende Ansicht haben:

<img width="1026" alt="Bildschirm­foto 2023-03-04 um 12 24 39" src="https://user-images.githubusercontent.com/114981111/222899096-288a7653-71ba-4d75-882e-e85af2efc2a4.png">

Bei Bedarf können Sie das deutsche Sprachpaket herunterladen, in dem Sie in der unteren rechten Ecke auf _"Installieren und neustarten"_ klicken. Ansonsten behält Visual Studio Code Englisch als Standardsprache bei.

Nach der Installation des Sprachpaketes, sieht das Fenster so aus: 

<img width="1136" alt="Bildschirm­foto 2023-03-04 um 12 25 34" src="https://user-images.githubusercontent.com/114981111/222899832-691f9e87-05d2-4f4f-890b-dd7640e1af44.png">

Klicken Sie nun auf das unterste Symbol in der Leiste auf der linken Seite, um zu den Erweiterungen zu gelangen. Sie sollten dann diese Ansicht vorliegen haben.

<img width="1136" alt="Bildschirm­foto 2023-03-04 um 12 25 55" src="https://user-images.githubusercontent.com/114981111/222899947-da71eeac-cf53-4c61-add7-0a0896743056.png">
